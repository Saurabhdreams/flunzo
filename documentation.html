How to upgrade from v1.3.3 to v1.3.3.1

This update is the completion of the latest update (1.3.3), next version will be 1.3.4
  • First, It's recommended to make a backup of your database and files before you start upgrading.
  • After that, delete these 2 folders node_modules and vendor from your server.
  • Then upload update.zip to your server. Extract files and replace the old one.
  • Now copy the following code and paste it in your languages files
    For example English
    Open lang/en/messages.php file and paste it at the bottom of the file before ];

  • Now go to this link https://your-domain.com/update make sure to put your domain name, and that's it.
